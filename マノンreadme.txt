マノン口上りーどみー

●使い方
「era悪魔でメイド」本体に同梱されているものより新しい口上ファイルをera板などで
入手した場合、フォルダ構造をそのままに「era悪魔でメイド」本体フォルダにコピーしてください。

●コンセプト
せっかくだから超乳キャラ作ろうぜ！
という勢いのままに、原案者の嗜好を存分に注ぎ込んだ結果生まれた何かです。

●どんなこ？
まともに精力を奪えないために淫魔社会を追放された落第淫魔です。
破瓜するのが怖い、男性と会話するのが怖い、性的なことをするのが好きじゃないと、
明らかに淫魔ではない別の何かであるべき人格を持って生まれた彼女ですが……その体には、
淫魔としてのありとあらゆる資質が完璧なまでに築き上げられていました。

すごくお堅く感じるのは表面上であり、中身は根っからの淫魔です。
かわいがってやってください。

●制限と解放
処女である彼女は最初、本番したがりません。
ですが、じっくりと体の感度を高め、精液と精子の味を覚えさせるか、あるいは焦がれさせるかする
ことによって、家主であるあなたにそのすべてをささげてくれます。
もっとも、あなたの理性の崩壊を止めるような野暮はありませんが……
その後の彼女の変貌は、まさに淫魔らしい淫魔のそれといえるでしょう。
